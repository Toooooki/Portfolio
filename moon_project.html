Moon Viewing WebApp

「月を身近に」するための現在位置・満ち欠け可視化アプリ

Technologies Used

Next.js TypeScript Tailwind CSS

Project Info

Role: Solo Developer
Period: 2026.1 - Present
Type: Personal Project

Overview / 背景

「夜」や「月」が持つ静かな自由に惹かれ（『よふかしのうた』に影響を受け）、 ふと月を見たいと思った時に、天気や方角、月齢といった情報が分散していることに不便を感じていました。 本プロジェクトは、「月を身近に」をコンセプトに、現在地・時刻・天候情報（OpenWeatherMap API）・天文学的計算（SunCalc）を統合し、アプリを開いた瞬間に「今、月がどう見えるか」を1画面で可視化するアプリケーションです。

Technical Highlights / 技術的なこだわり

1. クライアントサイドでのリアルタイム天文計算
月の満ち欠け画像を用意するのではなく、Canvas APIを使用して `arc()` メソッド等で影の部分を動的に描画しています。これにより、画像読み込みの遅延をなくし、理論上無限の解像度で月齢を表現可能です。

2. 3D表現とパフォーマンスの両立（開発中）
現在、React Three Fiber を用いた月の3D描画を実装中です。 単なる球体ではなく、シェーダーを用いて「夜の雰囲気」や「月明かり」を表現しつつ、 モバイル端末でも軽量に動作するようポリゴン数やライティングの最適化を行う予定です。

3. 開発環境の最適化
Windows環境特有のパス長制限やファイルシステムの差異による、本番デプロイ時の予期せぬエラーを防ぐため、 WSL2 (Ubuntu 22.04 LTS) 上に完全なLinux開発環境を構築しました。 Node.jsのバージョン管理には Volta を採用し、ローカル開発環境とCI/CD環境（Vercel）でのランタイムバージョンの不整合を排除しています。 これにより、コードを書くことだけに集中できるストレスフリーなデプロイフローを実現しました。
